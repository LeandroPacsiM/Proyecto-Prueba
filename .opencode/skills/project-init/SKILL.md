## What I do
- Analyze project requirements and suggest appropriate tech stack
- Create initial directory structure for web projects
- Set up basic configuration files (package.json, gitignore, etc.)
- Initialize Git repository with proper branches if not exists
- Create base HTML/CSS/JS files with modern standards

## When to use me
Use this skill when starting a new project from an empty or nearly empty repository.

## Project structure created

```
project/
├── .gitignore
├── package.json
├── index.html
├── src/
│   ├── index.js
│   └── styles/
│       └── main.css
└── README.md
```

## Guidelines
1. Always ask about project purpose before suggesting stack
2. Prefer vanilla JS for simple projects, frameworks for complex ones
3. Include accessibility basics from the start
4. Set up semantic HTML5 structure
5. Add responsive viewport meta tag
