## What I do
- Create standard portfolio sections (hero, about, projects, skills, contact)
- Implement responsive layouts that work on all devices
- Add smooth scrolling and subtle animations
- Include placeholder content that's easy to customize
- Ensure fast loading and good performance scores

## When to use me
Use this skill when building a personal portfolio or professional showcase website.

## Standard sections

### Hero Section
- Full viewport height on desktop
- Name, title, and brief tagline
- Call-to-action buttons (View Work, Contact)
- Optional: profile image or illustration

### About Section
- 2-3 paragraph bio
- Skills/technologies grid
- Optional: timeline or experience highlights

### Projects Section
- Grid of project cards (3 columns desktop, 1 mobile)
- Each card: title, description, tech tags, links (demo/repo)
- Hover effects for interactivity

### Skills Section
- Categorized skills (Frontend, Backend, Tools)
- Visual representation (tags, progress bars, or icons)
- Only include genuine proficiency levels

### Contact Section
- Email link with mailto:
- Social media links (GitHub, LinkedIn, Twitter)
- Optional: contact form with basic validation

## Design guidelines
- Consistent color scheme (2-3 primary colors + neutrals)
- Typography: 2 fonts max (one for headings, one for body)
- Whitespace: generous padding and margins
- Images: optimize for web (WebP format, lazy loading)
